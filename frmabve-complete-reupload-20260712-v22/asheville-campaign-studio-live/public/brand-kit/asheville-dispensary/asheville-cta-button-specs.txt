Asheville Dispensary CTA Button Specs

Use this setup to recreate the standard Asheville CTA button in Canva.

Canvas
- Size: 878 x 230 px
- Background color: #f3eee6

Button
- Size: 685 x 108 px
- Position: X 104 px, Y 62 px
- Fill color: #4a6f2f
- Corner radius: 2-3 px
- Effects: none
- Border: none

Text
- Example: Shop All THCA Flower
- Font: Montserrat SemiBold or Bold
- Size: 31-34 px
- Color: #ffffff
- Letter spacing: 4-6% in Canva
- Alignment: centered horizontally and vertically

Notes
- Keep the button flat and rectangular.
- Do not use black backgrounds for Asheville campaign CTAs.
- Use #263e22 only for dark green supporting sections, not this primary CTA fill.
